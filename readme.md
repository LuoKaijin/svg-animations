## 操作步骤

npm install

npm run gendata    

npm run build      

npm run start     


## 项目结构
    - config    配置文件
        - webpack.config.js  webpack打包配置文件
    - doc       工程相关的文档资料
    - src       工程资源文件
        - demo      相关案例说明 
        - src       静态文件(css,images等)
        - index.html    首页
        - index.js      首页脚本
        - template.html 首页需要的模板文件
    - build     打包文件输出目录
    - node.js   Node脚本,输出首页文件目录
    - .gitignore    git提交被忽略文件
    - package.json
    - readme.md
